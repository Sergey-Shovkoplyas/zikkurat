# Start template

Start template with (optional)
- normalize.css
- bootstrap-grid.min.css
- jquery-3.4.1.min.js

Usage 
- git clone(url)
- npm i      - install packeges
- gulp       - start development project
- gulp dist  - start production project(add autoprefixer, minify css....)

Features 
- sass 7-1 pattern (partial use)
- gulp task manager.
